N-Queens Problem

Placing N chess queens on an N×N chessboard so that no two queens threaten each other.

Time Complexity O(N!)
Space Complexity O(N²)

🎯 About N-Queens

The N-Queens puzzle is the problem of placing N chess queens on an N×N chessboard so that no two queens threaten each other; thus, a solution requires that no two queens share the same row, column, or diagonal.

🚀 The Approach

1

Start in the leftmost column

Begin by trying to place a queen in the first column.

2

Check Row Conflicts

Try placing a queen in each row of the current column. Check if the queen can be placed safely (no other queens in the same row or diagonals).

3

Recur

If a queen is placed safely, move to the next column and repeat step 2.

4

Backtrack

If no row is safe in the current column, go back to the previous column and try the next row for that queen.
