Rabin-Karp Algorithm

A string-searching algorithm that uses hashing to find any one of a set of pattern strings in a text.

Time Complexity Average O(n + m), Worst O(n * m)
Space Complexity O(1)

🎯 About Rabin-Karp Algorithm

The Rabin-Karp algorithm is a string-searching algorithm that uses hashing to find an exact match of a pattern string in a text. It was created by Richard M. Karp and Michael O. Rabin.

Unlike the naive approach, which checks for a match at every position, Rabin-Karp computes a hash value for the pattern and for each substring of the text of the same length. Only if the hash values match does it check the characters one by one.

🚀 The Approach

1

Calculate Hash

Calculate the hash value of the pattern and the first window of text (of length equal to the pattern).

2

Compare Hashes

Slide the pattern over text one by one. If the hash values of current window of text and pattern match, check the individual characters.

3

Exact Match Check

If the hash values match and characters also match one by one, pattern is found.

4

Rolling Hash

If hash values don't match, calculate hash value for next window of text using a rolling hash technique: remove the leading character and add the trailing character.
