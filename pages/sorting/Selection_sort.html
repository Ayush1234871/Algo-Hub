Selection Sort

Building a sorted sequence by repeatedly extracting the smallest element.

Best Case O(n²)
Average Case O(n²)
Space O(1)

🎯 About Selection Sort

Selection sort is an in-place comparison-based sorting algorithm. It has an O(n²) time complexity, which makes it inefficient on large lists, and generally performs worse than the similar insertion sort.

Selection sort is noted for its simplicity and has performance advantages over more complicated algorithms in certain situations, particularly where auxiliary memory is limited.

🚀 The Approach

1

Search for Minimum

Find the smallest element in the unsorted portion of the array.

2

Swap

Swap the found minimum element with the first element of the unsorted portion.

3

Mark as Sorted

Consider the swapped element as part of the sorted portion and the boundary moves one element right.

4

Repeat

Repeat the process for the remaining unsorted elements until the entire array is sorted.
