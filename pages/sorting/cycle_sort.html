Cycle Sort

An in-place sorting algorithm that minimizes memory writes by cyclically rotating elements.

Best Case O(n²)
Average Case O(n²)
Space O(1)

🎯 About Cycle Sort

Cycle sort is an in-place, unstable sorting algorithm, a comparison sort that is theoretically optimal in terms of the total number of writes to the original array.

It is based on the idea that the permutation to be sorted can be factored into cycles, which can each be rotated to give a sorted result. Unlike nearly every other sort, items are never written elsewhere in the array simply to push them out of the way of the action.

🚀 The Approach

1

Count Smaller

For each element, count how many elements in the rest of the array are smaller than it.

2

Find Position

If the element is not at its correct position (count), move it there. This is the start of a cycle.

3

Rotate Cycle

If there's another element already at that position, pick it up and repeat the process to find its correct position, continuing until the cycle returns to the starting index.

4

Iterate

Process all elements as potential cycle starts until the entire array is sorted.
