Quick Sort

High-performance sorting using the power of partitioning and recursion.

Best Case O(n log n)
Average Case O(n log n)
Space O(log n)

🎯 About Quick Sort

Quicksort is an efficient, comparison-based sorting algorithm that uses a divide-and-conquer strategy. It picks an element as a 'pivot' and partitions the given array around the picked pivot.

Because it is an in-place sort (it doesn't require extra storage like Merge Sort), it is often the preferred choice for large datasets in real-world applications.

🚀 The Approach

1

Pick a Pivot

Select an element from the array to be the pivot (often the last element or a random one).

2

Partition

Reorder the array so that all elements with values less than the pivot come before it, and all elements with values greater than the pivot come after it.

3

Recurse

Recursively apply the same steps to the sub-array of smaller elements and the sub-array of larger elements.

4

Base Case

The recursion stops when the sub-array has zero or one elements, as they are already sorted.
