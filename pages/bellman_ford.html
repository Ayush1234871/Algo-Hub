Bellman-Ford Algorithm

Find shortest paths from a single source vertex to all other vertices, even with negative edge weights.

Time Complexity O(V * E)
Space Complexity O(V)

🎯 About Bellman-Ford

The Bellman-Ford algorithm computes shortest paths from a single source vertex to all of the other vertices in a weighted digraph. It is slower than Dijkstra's algorithm for the same problem, but more versatile, as it is capable of handling graphs in which some of the edge weights are negative numbers.

Importantly, the Bellman-Ford algorithm can detect negative cycles in a graph, which is useful because a shortest path is not defined if the graph contains a negative cycle.

🚀 The Approach

1

Initialize

Initialize distance to source as 0 and to all other vertices as infinity.

2

Relax Edges

Repeat the following step |V| - 1 times (where |V| is the number of vertices): For each edge (u, v) with weight w, if dist[u] + w < dist[v], then dist[v] = dist[u] + w.

3

Check Negative Cycles

Iterate through all edges one more time. If dist[u] + w < dist[v] for any edge, then the graph contains a negative weight cycle.
