Heap Sort

Utilizing the Binary Heap data structure for efficient, in-place sorting.

Best Case O(n log n)
Average Case O(n log n)
Space O(1)

🎯 About Heap Sort

Heap Sort is a comparison-based sorting technique based on a Binary Heap data structure. It is similar to selection sort where we first find the maximum element and place the maximum element at the end.

We repeat the same process for the remaining elements. It is an in-place algorithm, but its typical implementation is not stable.

🚀 The Approach

1

Build Max Heap

Construct a max heap from the input data.

2

Swap Root

At this point, the greatest item is stored at the root of the heap. Replace it with the last item of the heap followed by reducing the size of the heap by 1.

3

Heapify

Heapify the root of the tree to maintain the max-heap property.

4

Repeat

Repeat steps 2 and 3 while the size of the heap is greater than 1.
