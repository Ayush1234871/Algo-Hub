Ternary Search

Dividing the search space into three equal parts to find targets in sorted arrays.

Best Case O(1)
Average Case O(log₃ n)
Space O(1)

🎯 About Ternary Search

Ternary search is a decrease-and-conquer algorithm that find the position of a target value within a sorted array. It is similar to binary search but instead of dividing the array into two parts, it divides it into three equal parts.

While it requires more comparisons per step than binary search, it reduces the search space by a factor of 3 in each iteration.

🚀 The Approach

1

Initialize

Begin with a sorted array and set left to 0 and right to n-1.

2

Calculate Midpoints

Find two midpoints: mid1 = l + (r-l)/3 and mid2 = r - (r-l)/3.

3

Compare

Check if arr[mid1] == target or arr[mid2] == target. If yes, return the index.

4

Narrow Range

If target is smaller than arr[mid1], search in the left third. If target is larger than arr[mid2], search in the right third. Else, search in the middle third.

5

Loop

Repeat the process until the target is found or left > right.
