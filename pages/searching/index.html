Searching Algorithms

Linear Search

The simplest search algorithm that checks every element in order.

Binary Search

Efficiently find elements in sorted arrays by repeatedly dividing the search interval in half.

Exponential Search

Finds the range where the element is likely to be and performs binary search.

Ternary Search

Similar to binary search but divides the array into three parts.
