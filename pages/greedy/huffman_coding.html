Huffman Coding

An optimal prefix code algorithm used for lossless data compression.

Time Complexity O(n log n)
Space O(n)

🎯 About Huffman Coding

Huffman coding is a lossless data compression algorithm. The idea is to assign variable-length codes to input characters, lengths of the assigned codes are based on the frequencies of corresponding characters.

The most frequent character gets the smallest code and the least frequent character gets the largest code. It uses a priority queue to build a binary tree where leaves represent characters.

🚀 The Approach

1

Create Nodes

Create a leaf node for each unique character and build a min heap of all leaf nodes.

2

Extract Min

Extract two nodes with the minimum frequency from the min heap.

3

Combine

Create a new internal node with a frequency equal to the sum of the two nodes' frequencies. Make the first extracted node as its left child and the other as its right child. Add this node to the min heap.

4

Repeat

Repeat steps 2 and 3 until the heap contains only one node. The remaining node is the root node and the tree is complete.
