Insertion Sort

Sorting like a hand of playing cards: inserting one element at a time into its correct place.

Best Case O(n)
Average Case O(n²)
Space O(1)

🎯 About Insertion Sort

Insertion sort is a simple sorting algorithm that builds the final sorted array one item at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.

However, insertion sort provides several advantages: it is easy to implement, efficient for small datasets, and adaptive (efficient for datasets that are already substantially sorted).

🚀 The Approach

1

Initialize

Assume the first element is already sorted. Pick the next element.

2

Compare

Compare the picked element with the elements in the sorted sub-list (to its left).

3

Shift

Shift all the elements in the sorted sub-list that are greater than the picked element to be moved to the right.

4

Insert

Insert the element into the correct position.

5

Repeat

Repeat the process for all subsequent elements until the entire list is sorted.
