Division Hashing

The fundamental hashing method using modular arithmetic for efficient indexing.

Average Case O(1)
Worst Case O(n)
Space O(n)

🎯 About Division Hashing

Division hashing is one of the simplest and most commonly used hashing techniques. It works by taking the remainder of a division operation between the key and the hash table size. The formula is: h(k) = k mod m, where k is the key and m is the size of the table.

Choosing a prime number for m that is not too close to a power of 2 is often recommended to minimize collisions and ensure a more uniform distribution of keys.

🚀 The Approach

1

Input Key

Receive the numerical key k that needs to be hashed into the table.

2

Determine Table Size

Set the hash table size m. A prime number is preferred for better distribution.

3

Apply Formula

Calculate the hash value using the modulo operation: index = k % m.

4

Return Index

Use the resulting index to store or retrieve the data in the hash table.
