Depth-First Search (DFS)

Traverse or search tree and graph data structures by exploring as far as possible along each branch.

Time Complexity O(V + E)
Space Complexity O(V)

🎯 About DFS

Depth-First Search (DFS) is an algorithm for traversing or searching tree or graph data structures. The algorithm starts at the root node (selecting some arbitrary node as the root node in the case of a graph) and explores as far as possible along each branch before backtracking.

It can be implemented easily using recursion or a stack data structure.

🚀 The Approach (Recursive)

1

Initialize

Create a boolean array to keep track of visited vertices.

2

Visit

Start from a given vertex, mark it as visited and print it.

3

Recurse

For every adjacent vertex to the current vertex, if it has not been visited, recursively call the DFS function for that vertex.

4

Backtrack

When a vertex has no unvisited adjacent vertices, the algorithm naturally backtracks to the previous vertex.
