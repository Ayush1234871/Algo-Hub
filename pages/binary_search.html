Binary Search

An efficient algorithm for finding an item from a sorted list of items.

Best Case O(1)
Average Case O(log n)
Space O(1)

🎯 About Binary Search

Binary Search is a search algorithm that finds the position of a target value within a sorted array. It compares the target value to the middle element of the array.

If they are not equal, the half in which the target cannot lie is eliminated and the search continues on the remaining half, again taking the middle element to compare to the target value, and repeating this until the target value is found.

🚀 The Approach

1

Initialize

Begin with a sorted array. Set the low pointer to 0 and high pointer to the last index.

2

Find Middle

Calculate the midpoint: mid = low + (high - low) / 2.

3

Compare

If array[mid] == target, return mid. If array[mid] < target, set low = mid + 1. Otherwise, set high = mid - 1.

4

Repeat

Repeat steps 2 and 3 until low > high.

5

Not Found

If the search terminates without a match, the target is not in the array. Return -1.
