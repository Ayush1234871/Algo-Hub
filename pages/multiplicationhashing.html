Multiplication Hashing

An efficient hashing technique that uses fractional multiplication to distribute keys uniformly.

Average Case O(1)
Worst Case O(n)
Space O(n)

🎯 About Multiplication Hashing

Multiplication hashing is a method where the key k is multiplied by a constant A (where 0 < A < 1), and the fractional part of k * A is extracted. This fractional part is then multiplied by the table size m, and the floor of the result is taken as the hash value.

The formula is: h(k) = ⌊m * (k * A mod 1)⌋. A popular value for A is (√5 - 1) / 2 ≈ 0.6180339887... (the Golden Ratio), which helps in achieving a very uniform distribution.

🚀 The Approach

1

Choose Constant

Select a constant A such that 0 < A < 1. The Golden Ratio is a common choice.

2

Multiply Key

Multiply the key k by the constant A.

3

Extract Fractional Part

Extract the fractional part of the product: (k * A) mod 1.

4

Map to Table

Multiply the fractional part by table size m and take the floor to get the index.
