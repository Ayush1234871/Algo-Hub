Exponential Search

Optimizing search in large or infinite sorted datasets by doubling search ranges.

Best Case O(1)
Average Case O(log n)
Space O(1)

🎯 About Exponential Search

Exponential search (also known as doubling search or galloping search) is an algorithm created for searching large sorted arrays. It works by finding an interval where the value might exist and then performing a binary search within that range.

It is particularly effective if the element is near the beginning of the array or when the size of the array is unknown (infinite lists).

🚀 The Approach

1

Initial Check

Check if the first element is the target. If yes, return index 0.

2

Find Range

Starting with index 1, keep doubling the index (1, 2, 4, 8...) as long as the element at the index is less than the target and within array bounds.

3

Range Identified

Once you find an index i such that arr[i] > target, the target must be between i/2 and min(i, n-1).

4

Binary Search

Perform a standard binary search within the small sub-range identified in step 3.
