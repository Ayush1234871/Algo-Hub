Radix Sort

A non-comparative sorting algorithm that sorts data with integer keys by grouping keys by digits.

Best Case O(nk)
Average Case O(nk)
Space O(n+k)

🎯 About Radix Sort

Radix sort is an efficient non-comparative sorting algorithm. It avoids comparison by creating and distributing elements into buckets according to their radix (base). For elements with more than one significant digit, this bucketing process is repeated for each digit.

It is often used to sort large numbers or strings where the length of the keys (k) is small relative to the number of elements (n).

🚀 The Approach

1

Find Maximum

Identify the maximum element in the array to determine the number of digits (k).

2

Digit Loop

Process each digit position starting from the least significant digit (LSD) up to the most significant digit (MSD).

3

Bucket Distribution

For the current digit position, distribute the elements into 10 buckets (0-9).

4

Collect

Re-collect the elements from the buckets in order. The array is now sorted relative to the current digit.

5

Completion

After processing all digit positions, the array is fully sorted.
