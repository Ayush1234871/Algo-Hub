Linear Search

The most straightforward approach to finding an element in a collection.

Best Case O(1)
Average Case O(n)
Space O(1)

🎯 About Linear Search

Linear search (or sequential search) is the simplest method for finding a target value within a list. It checks each element of the list sequentially until a match is found or the entire list has been searched.

This algorithm is highly versatile as it does not require the data to be sorted, making it the go-to choice for small or unordered datasets.

🚀 The Approach

1

Initialize

Start from the leftmost element of the array (index 0).

2

Compare

Compare the current element with the target value you're looking for.

3

Match Found

If a match is found, return the current index and terminate the search.

4

Next Element

If no match, move to the next index and repeat the comparison.

5

Not Found

If the end of the array is reached without a match, return -1.
