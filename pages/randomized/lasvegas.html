🎯 About Las Vegas Algorithms

A Las Vegas algorithm is a randomized algorithm that always produces the correct result or informs about failure. The randomness affects only the resources used (primarily execution time). The name "Las Vegas" refers to the fact that the algorithm "gambles" with time but never with correctness.

Contrast this with Monte Carlo algorithms, where the running time is fixed but the result may have a non-zero probability of being incorrect.

🚀 The Approach

1

Introduce Randomness

Identify where a random choice can be made (e.g., picking a pivot in QuickSort).

2

Execute Logic

Run the algorithm using the random selections to guide its path.

3

Verify Termination

The algorithm continues until the correct result is found.

4

Probabilistic Time

The expected running time is typically efficient, even if the worst-case time is high.
