Hashing Algorithms

Division Hashing

The simplest hashing method using the modulo operator to map keys to slots.

Multiplication Hashing

A more flexible hashing technique that uses fractional parts to distribute keys.

Universal Hashing

A randomized approach to hashing that minimizes collisions through function selection.

SHA-256

A cryptographic hash function that provides high security and data integrity.
