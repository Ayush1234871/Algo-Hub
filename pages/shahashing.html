🎯 About SHA-256

SHA-256 (Secure Hash Algorithm 256-bit) is a cryptographic hash function that produces a fixed-size 256-bit (32-byte) hash value. It is widely used in security applications, including digital signatures, SSL certificates, and blockchain technology (like Bitcoin).

It is designed to be one-way, meaning it is computationally infeasible to reverse the hash to find the original input. It is also collision-resistant, making it highly unlikely for two different inputs to produce the same hash.

🚀 The Approach

1

Preprocessing

Pad the input message to ensure its length is a multiple of 512 bits and append the original bit length.

2

Initialize Hash Values

Set up eight 32-bit words as the initial hash values (Initialization Vector).

3

Compression Loop

Divide the message into 512-bit blocks and process each block through 64 rounds of logical operations.

4

Final Output

Concatenate the final hash words into a single 256-bit string, usually represented in hexadecimal.
