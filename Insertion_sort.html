Insertion Sort

About the Insertion Sort

Insertion Sort is a simple sorting algorithm that builds the final sorted array one element at a time, inserting each element into its correct position. It is efficient for small datasets and often performs well for partially sorted arrays.

Approach

To solve the Insertion Sort problem:

  1. Initialize: Start with the second element (index 1) assuming the first element (index 0) is already sorted.
  2. Compare: Compare the current element with the previous elements in the sorted portion. Move elements greater than the current one to the right.
  3. Insert: Insert the current element into its correct position in the sorted portion by shifting larger elements to the right.
  4. Repeat: Repeat steps 2-3 for the remaining unsorted elements, gradually expanding the sorted portion.
  5. Completion: The array is now sorted when all elements have been included in the sorted portion.
