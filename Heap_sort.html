Heap Sort

About Heap Sort

HeapSort is a comparison-based sorting algorithm that uses a binary heap data structure to build a max-heap or min-heap and efficiently sort elements.

Approach

To implement Heap Sort:

  • Build Heap: Build a max-heap from the array.
  • Heapify: Swap the root (maximum element) with the last element, reduce the heap size, and heapify the root.
  • Sort: Repeat the heapify and swap steps until the heap is empty, resulting in a sorted array.
