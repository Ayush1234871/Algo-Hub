Exponential Search Algorithm

About the Sential Linear Search

Exponential search, also known as galloping search or doubling search, is an algorithm used to find an element in a sorted array. It begins with a small range and repeatedly doubles the size of the search interval until an element greater than the target is found. Once the range is identified, it performs a binary search within that interval for the specific target element. Exponential search is particularly efficient for large datasets, providing a time complexity of O(log n).

Approach

To solve the Exponential search problem:

  1. Start with the first element of the array as the initial search interval's lower bound and set the upper bound to the minimum of either the array size or a power of 2..
  2. Double the size of the search interval iteratively until an element greater than the target is encountered or the end of the array is reached.
  3. Perform a binary search within the identified interval to locate the exact position of the target element.
  4. Choose an optimal doubling factor based on the dataset size, ensuring a balance between the reduced number of iterations and the overhead of repeatedly doubling the interval.
  5. Exponential search provides a time complexity of O(log n), making it efficient for large datasets compared to linear search. The initial exponential growth phase quickly narrows down the search space before the final binary search refines the result.
